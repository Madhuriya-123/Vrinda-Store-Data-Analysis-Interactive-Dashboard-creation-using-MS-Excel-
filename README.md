# 📊 Vrinda Store Data Analysis & Interactive Dashboard using MS Excel

## 📝 Project Overview

This project focuses on analyzing the annual sales data of **Vrinda Store** using **Microsoft Excel**.
The objective is to build an **interactive dashboard** that helps the business understand customer behavior, sales trends, and overall performance for better decision-making.

The dashboard was created using:

* Data Cleaning
* Data Processing
* Pivot Tables
* Pivot Charts
* Slicers
* Data Visualization Techniques

---

# 🎯 Project Objective

Vrinda Store wants to create an annual sales report for **2022** to understand:

* Customer purchasing behavior
* Monthly sales and order trends
* Best-performing states and channels
* Customer demographics
* Product category performance

This analysis helps the business improve sales strategies for **2023 & 2024**.

# 📌 Business Questions (KPIs)

The dashboard answers the following questions:

1. Compare Sales and Orders using a single chart
2. Which month got the highest sales and orders?
3. Who purchased more — Men or Women?
4. What are the different order statuses in 2022?
5. Top states contributing to sales
6. Relationship between age and gender based on number of orders
7. Which sales channel contributes the most?
8. Highest-selling category
9. Percentage of total orders delivered

# 📈 Project Insights

* Women customers contribute approximately **64%** of total sales.
* Maharashtra, Karnataka, and Uttar Pradesh are the top-performing states.
* Adult customers aged **30–49 years** contribute the highest number of orders.
* Amazon, Flipkart, and Myntra are the leading sales channels.
* More than **90%** of products were successfully delivered.

---

# ✅ Final Conclusion

To improve sales performance, Vrinda Store should focus on:

* Women customers aged **30–49 years**
* Customers from Maharashtra, Karnataka, and Uttar Pradesh
* Marketing through Amazon, Flipkart, and Myntra platforms

Targeted promotions, personalized offers, and customer-focused marketing campaigns can help increase future sales and customer engagement.
